    해당 데이터가 있는 컬럼의 컬럼타입이 문자형인지 숫자형인지 확인 후 숫자형 컬럼타입으로 데이터 조건을 재 설정 후 진행해보시길 바랍니다. 문자형 컬럼인 경우 데이터가 숫자로 나타나더라도 연산이 되지 않습니다.
    *문자형 컬럼을 숫자형 컬럼으로 바꾸는 방법 문자형의 컬럼을 숫자형으로 바꾸기 위해서는 데이터 보정에서 컬럼 타입을 숫자형으로 바꿔 저장하면 됩니다.
